# Docker Deployment

When Carrion has a new release it is automatically uploaded to docker hub as well.

If you're running Podman or Docker you can pull it using the following:

## Latest Build
```bash

docker pull javanhut/carrionlanguage:latest
```

## Specific Version
 ```bash
docker pull javanhut/carrionlanguage:tagname
```
