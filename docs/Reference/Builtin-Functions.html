Built-in Functions Reference

Carrion provides a rich set of built-in functions for common operations. These functions are available globally without any imports.

I/O Functions

print()

print(value1, value2, ...)

Outputs values to the console. Multiple values are separated by spaces.

print("Hello, World!")
print("Name:", name, "Age:", age)
print(f"The answer is {42}")

input()

input(prompt) -> String

Reads a line of input from the user, displaying an optional prompt.

name = input("Enter your name: ")
age = int(input("Enter your age: "))

Type Conversion Functions

int()

int(value) -> Integer

Converts a value to an integer. Works with strings and floats.

x = int("42")      // 42
y = int(3.14)      // 3
z = int("10", 16)  // 16 (hex conversion)

float()

float(value) -> Float

Converts a value to a floating-point number.

x = float("3.14")  // 3.14
y = float(42)      // 42.0

str()

str(value) -> String

Converts any value to its string representation.

s = str(42)        // "42"
s = str([1, 2, 3]) // "[1, 2, 3]"

bool()

bool(value) -> Boolean

Converts a value to boolean. Empty strings, 0, None, and empty collections are False.

bool(1)       // True
bool(0)       // False
bool("hello") // True
bool("")      // False

Collection Functions

len()

len(collection) -> Integer

Returns the length of a string, list, or hash.

len("hello")     // 5
len([1, 2, 3])   // 3
len({"a": 1})    // 1

range()

range(start, end, step) -> List

Generates a list of numbers from start to end (exclusive).

range(5)         // [0, 1, 2, 3, 4]
range(1, 5)      // [1, 2, 3, 4]
range(0, 10, 2)  // [0, 2, 4, 6, 8]

append()

append(list, value) -> List

Adds an element to the end of a list and returns the modified list.

items = [1, 2, 3]
items = append(items, 4)  // [1, 2, 3, 4]

keys()

keys(hash) -> List

Returns a list of all keys in a hash.

data = {"name": "Carrion", "version": "0.1.9"}
keys(data)  // ["name", "version"]

values()

values(hash) -> List

Returns a list of all values in a hash.

data = {"a": 1, "b": 2}
values(data)  // [1, 2]

Character Functions

ord()

ord(char) -> Integer

Returns the ASCII/Unicode code point of a character.

ord("A")   // 65
ord("a")   // 97
ord("\n")  // 10

chr()

chr(code) -> String

Returns the character for an ASCII/Unicode code point.

chr(65)   // "A"
chr(97)   // "a"
chr(10)   // "\n"

Type Checking Functions

type()

type(value) -> String

Returns the type name of a value.

type(42)        // "INTEGER"
type("hello")   // "STRING"
type([1, 2])    // "ARRAY"
type({"a": 1})  // "HASH"

Math Functions

abs()

abs(number) -> Number

Returns the absolute value of a number.

abs(-5)    // 5
abs(3.14)  // 3.14
abs(-2.5)  // 2.5

min() / max()

min(a, b, ...) -> Number
max(a, b, ...) -> Number

Returns the minimum or maximum value from the arguments.

min(1, 5, 3)     // 1
max(1, 5, 3)     // 5
min([1, 2, 3])   // 1 (with list)

Utility Functions

sleep()

sleep(seconds)

Pauses execution for the specified number of seconds.

print("Starting...")
sleep(2)
print("Done after 2 seconds")

exit()

exit(code)

Terminates the program with the specified exit code.

if error:
    exit(1)
exit(0)  // Success
