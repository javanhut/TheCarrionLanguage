Keywords Reference

Carrion uses mystical terminology for many of its keywords, making code read like magical incantations. Here's a complete reference of all reserved keywords.

Definitions

Keyword Traditional Equivalent Description
spell def / function Defines a function (spell)
grim class Defines a class (grimoire)
init __init__ / constructor Constructor method for grimoires
self this Reference to the current instance
super super Reference to parent class

Control Flow

Keyword Traditional Equivalent Description
if if Conditional statement
elif else if Else-if branch
else else Default branch
for for For loop iteration
while while While loop
repeat for i in range(n) Repeat a block n times
break break Exit a loop
continue continue Skip to next iteration
return return Return a value from a spell

Error Handling

Keyword Traditional Equivalent Description
attempt try Begin error handling block
ensnare catch / except Catch an error
resolve finally Always execute cleanup
raise throw / raise Throw an error

Logical Operators

Keyword Description
and Logical AND
or Logical OR
not Logical NOT
in Membership test

Values

Keyword Description
True Boolean true value
False Boolean false value
None Null/nil value

Modules

Keyword Description
import Import a module or grimoire
from Selective import
as Import alias

Usage Examples

// Defining a spell (function)
spell calculate_power(base, multiplier):
    return base * multiplier

// Defining a grimoire (class)
grim Wizard:
    init(name, level):
        self.name = name
        self.level = level

    spell cast_fireball():
        return f"{self.name} casts Fireball!"

// Error handling
attempt:
    risky_operation()
ensnare MagicError as e:
    print(f"Error: {e}")
resolve:
    cleanup()

// Control flow
for item in items:
    if item == target:
        break

repeat 5:
    print("Casting spell...")
