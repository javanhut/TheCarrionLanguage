Step 3: Grimoire Crafting (Classes) (10 minutes)

Classes in Carrion are called ‚Äúgrimoires‚Äù (magical spellbooks):

// Basic grimoire
grim MagicalCreature:
    init(name, element, health = 100):
        self.name = name
        self.element = element
        self.health = health
        self.max_health = health
    
    spell introduce():
        return f"I am {self.name}, a creature of {self.element}!"
    
    spell take_damage(damage):
        self.health = max(0, self.health - damage)
        if self.health == 0:
            return f"üíÄ {self.name} has been defeated!"
        else:
            return f"üíî {self.name} takes {damage} damage! Health: {self.health}/{self.max_health}"
    
    spell heal(amount):
        old_health = self.health
        self.health = min(self.max_health, self.health + amount)
        healed = self.health - old_health
        return f"üíö {self.name} heals for {healed} HP! Health: {self.health}/{self.max_health}"

// Inheritance - specialized grimoires
grim Dragon(MagicalCreature):
    init(name, color):
        super.init(name, "fire", 200)  // Dragons have more health
        self.color = color
        self.breath_attack_power = 75
    
    spell breathe_fire():
        return f"üî• {self.color} dragon {self.name} breathes fire for {self.breath_attack_power} damage!"
    
    spell intimidate():
        return f"üò± {self.name} roars menacingly! All enemies are frightened!"

// Create and use magical creatures
unicorn = MagicalCreature("Stardust", "light", 80)
dragon = Dragon("Smaug", "Red")

print(unicorn.introduce())
print(dragon.introduce())
print(dragon.breathe_fire())
print(unicorn.take_damage(30))

Key Takeaway: Grimoires use grim keyword, methods are ‚Äúspells‚Äù, and inheritance works like Python with super!

grim BattleMage:

        self.element = element

        self.mana = 50
        self.spells = Array(["Magic Missile"])

            self.spells.append(spell_name)
            return f"üìö {self.name} learned {spell_name}!"
        return f"ü§î {self.name} already knows {spell_name}"
    
    spell cast_spell(spell_name, target):
        if not self.spells.contains(spell_name):
            return f"‚ùå {self.name} doesn't know {spell_name}!"
        
        if self.mana < 10:
            return f"üí´ {self.name} is out of mana!"
        
        self.mana -= 10
        damage = 25
        
        if spell_name == "Fireball" and self.element == "fire":
            damage *= 2  // Elemental bonus

        return f"‚ú® {self.name} casts {spell_name} on {target.name} for {damage} damage!"
