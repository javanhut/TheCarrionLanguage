Data Types and Variables

Carrion is dynamically typed, meaning variables don't have fixed types. This guide covers all the data types available in Carrion.

Primitive Types

Integer INTEGER

Whole numbers, positive or negative, without decimals.

age = 25
negative = -100
big_number = 1000000

// Arithmetic operations
sum = 10 + 5      // 15
diff = 10 - 3     // 7
product = 4 * 5   // 20
quotient = 20 / 4 // 5
power = 2 ** 10   // 1024
remainder = 17 % 5 // 2

Float FLOAT

Numbers with decimal points.

pi = 3.14159
temperature = -40.5
scientific = 1.5e10  // 15000000000.0

// Float arithmetic
result = 3.14 * 2    // 6.28
division = 10 / 3    // 3.333...

String STRING

Text data enclosed in quotes.

// Single or double quotes
name = "Carrion"
greeting = 'Hello, World!'

// Multi-line strings
poem = """
Roses are red,
Violets are blue,
Carrion is magical,
And so are you!
"""

// String operations
upper = name.upper()           // "CARRION"
lower = name.lower()           // "carrion"
length = len(name)             // 7
char = name[0]                 // "C"
last = name[-1]                // "n"
contains = "arr" in name       // True

// String formatting
message = f"Welcome to {name}!"
concat = "Hello, " + name + "!"

Boolean BOOLEAN

True or False values.

is_active = True
is_complete = False

// Boolean operations
result = True and False   // False
result = True or False    // True
result = not True         // False

// Comparisons return booleans
is_equal = 5 == 5         // True
is_greater = 10 > 5       // True

None NULL

Represents the absence of a value.

value = None

// Check for None
if value == None:
    print("No value set")

// Common use: default return
spell find_item(items, target):
    for item in items:
        if item == target:
            return item
    return None  // Not found

Collection Types

Array (List) ARRAY

Ordered, mutable collection of items.

// Creating arrays
numbers = [1, 2, 3, 4, 5]
mixed = [1, "two", 3.0, True]
empty = []

// Accessing elements
first = numbers[0]      // 1
last = numbers[-1]      // 5
slice = numbers[1:3]    // [2, 3]

// Modifying arrays
numbers[0] = 10
numbers = append(numbers, 6)

// Array operations
length = len(numbers)
contains = 3 in numbers
combined = [1, 2] + [3, 4]  // [1, 2, 3, 4]

// Iteration
for num in numbers:
    print(num)

Hash (Dictionary) HASH

Key-value pairs for structured data.

// Creating hashes
person = {
    "name": "Merlin",
    "age": 150,
    "magic": True
}

// Accessing values
name = person["name"]

// Modifying hashes
person["age"] = 151
person["title"] = "Wizard"

// Hash operations
all_keys = keys(person)     // ["name", "age", "magic", "title"]
all_values = values(person)
has_key = "name" in person  // True

// Iteration
for key in keys(person):
    print(f"{key}: {person[key]}")

Tuple TUPLE

Immutable ordered collection.

// Creating tuples
coordinates = (10, 20)
rgb = (255, 128, 0)

// Accessing elements
x = coordinates[0]  // 10
y = coordinates[1]  // 20

// Tuple unpacking
x, y = coordinates

// Use in functions for multiple returns
spell get_min_max(numbers):
    return (min(numbers), max(numbers))

minimum, maximum = get_min_max([3, 1, 4, 1, 5])

Advanced Data Structures

BTree BTREE

Binary search tree for sorted data.

// Create and populate
tree = BTree()
tree.insert(10)
tree.insert(5)
tree.insert(15)

// Operations
size = tree.get_size()    // 3 (renamed from size() in v0.1.9)
found = tree.search(10)   // True
tree.display_tree()       // Visual tree display (new in v0.1.9)

// Traversals
inorder = tree.inorder()  // Sorted order

Stack STACK

Last-In-First-Out (LIFO) data structure.

stack = Stack()
stack.push(1)
stack.push(2)
stack.push(3)

top = stack.peek()    // 3
item = stack.pop()    // 3
size = stack.get_size()   // 2
empty = stack.is_empty()  // False

Queue QUEUE

First-In-First-Out (FIFO) data structure.

queue = Queue()
queue.enqueue("first")
queue.enqueue("second")
queue.enqueue("third")

front = queue.peek()      // "first"
item = queue.dequeue()    // "first"
size = queue.get_size()   // 2

Type Conversion

// String to number
num = int("42")       // 42
decimal = float("3.14")  // 3.14

// Number to string
text = str(42)        // "42"
text = str(3.14)      // "3.14"

// To boolean
bool(1)        // True
bool(0)        // False
bool("")       // False
bool("hello")  // True

// Check type
type(42)       // "INTEGER"
type("hello")  // "STRING"
type([1, 2])   // "ARRAY"

Variables

// Variable assignment
name = "Carrion"
count = 0

// Multiple assignment
x, y, z = 1, 2, 3

// Reassignment (dynamic typing)
value = 42        // Integer
value = "forty-two"  // Now a string

// Naming conventions
my_variable = 1        // snake_case (recommended)
myVariable = 2         // camelCase (also valid)
MY_CONSTANT = 3.14     // UPPER_CASE for constants
