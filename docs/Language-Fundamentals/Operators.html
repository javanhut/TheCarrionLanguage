Operators and Expressions

Carrion supports a variety of operators for arithmetic, comparison, logical operations, and more.

Arithmetic Operators

OperatorNameExampleResult
+Addition5 + 38
-Subtraction10 - 46
*Multiplication6 * 742
/Division15 / 35.0
%Modulo17 % 52
**Exponentiation2 ** 101024
// Arithmetic examples
sum = 10 + 5           // 15
difference = 20 - 8     // 12
product = 4 * 5         // 20
quotient = 15 / 4       // 3.75
remainder = 17 % 5      // 2
power = 2 ** 8          // 256

// Integer division (floor)
whole = int(17 / 5)     // 3

Comparison Operators

OperatorNameExampleResult
==Equal5 == 5True
!=Not equal5 != 3True
<Less than3 < 5True
>Greater than5 > 3True
<=Less than or equal5 <= 5True

// Comparison examples
is_equal = 10 == 10      // True
not_equal = 10 != 5      // True
less_than = 3 < 5        // True
greater = 10 > 5         // True

// Chained comparisons
x = 5
in_range = 1 < x < 10    // True (equivalent to: 1 < x and x < 10)

// String comparison (lexicographic)
"apple" < "banana"       // True
"abc" == "abc"           // True

Logical Operators

OperatorNameDescription
andLogical ANDTrue if both operands are true
orLogical ORTrue if at least one operand is true
notLogical NOTInverts the boolean value
// Logical AND
True and True    // True
True and False   // False
False and True   // False
False and False  // False

// Logical OR
True or True     // True
True or False    // True
False or True    // True
False or False   // False

// Logical NOT
not True         // False
not False        // True

// Practical examples
age = 25
has_license = True

can_drive = age >= 18 and has_license  // True

is_weekend = False
is_holiday = True
day_off = is_weekend or is_holiday      // True

is_working = not day_off                // False

Membership Operators

OperatorDescriptionExample
inTrue if value is in sequence"a" in "apple"
not inTrue if value is not in sequence5 not in [1, 2, 3]
// String membership
"car" in "Carrion"         // True
"xyz" not in "Carrion"     // True

// List membership
fruits = ["apple", "banana", "cherry"]
"apple" in fruits          // True
"grape" in fruits          // False
"grape" not in fruits      // True

// Hash key membership
person = {"name": "Merlin", "age": 150}
"name" in person           // True (checks keys)
"address" not in person    // True

Assignment Operators

OperatorExampleEquivalent

String Operators

// Concatenation
greeting = "Hello, " + "World!"  // "Hello, World!"

// Repetition
line = "-" * 20                  // "--------------------"

// Membership
has_at = "@" in "user@email.com" // True

// Indexing
text = "Carrion"
first = text[0]     // "C"
last = text[-1]     // "n"

// Slicing
sub = text[0:3]     // "Car"
sub = text[3:]      // "rion"
sub = text[:3]      // "Car"

Operator Precedence

From highest to lowest precedence:

PrecedenceOperators
1 (Highest)** (exponentiation)

// Precedence examples
result = 2 + 3 * 4      // 14 (not 20)
result = (2 + 3) * 4    // 20 (parentheses override)

result = 2 ** 3 ** 2    // 512 (right-to-left for **)
result = (2 ** 3) ** 2  // 64

// Logical precedence
x = True or False and False  // True (and evaluates first)
x = (True or False) and False // False
