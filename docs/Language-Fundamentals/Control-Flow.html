Control Flow

Control flow statements determine the order in which code is executed. Carrion provides familiar constructs with some magical twists.

Conditional Statements

If / Elif / Else

Execute code based on conditions:

// Basic if
if temperature > 30:
    print("It's hot!")

// If-else
if is_raining:
    print("Take an umbrella")
else:
    print("Enjoy the sun")

// If-elif-else chain
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Your grade: {grade}")

Nested Conditions

if user.is_logged_in:
    if user.is_admin:
        print("Welcome, Administrator!")
    else:
        print(f"Welcome, {user.name}!")
else:
    print("Please log in")

Compound Conditions

// Using 'and'
if age >= 18 and has_license:
    print("You can drive")

// Using 'or'
if is_weekend or is_holiday:
    print("No work today!")

// Using 'not'
if not is_complete:
    print("Still working...")

// Membership test with 'in'
if "admin" in user.roles:
    print("Admin access granted")

Loops

For Loops

Iterate over sequences:

// Iterate over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

// Iterate with range
for i in range(5):
    print(i)  // 0, 1, 2, 3, 4

// Range with start and end
for i in range(1, 6):
    print(i)  // 1, 2, 3, 4, 5

// Range with step
for i in range(0, 10, 2):
    print(i)  // 0, 2, 4, 6, 8

// Iterate over hash keys
person = {"name": "Merlin", "age": 150}
for key in keys(person):
    print(f"{key}: {person[key]}")

While Loops

Loop while a condition is true:

// Basic while loop
count = 0
while count < 5:
    print(count)
    count = count + 1

// With user input
password = ""
while password != "secret":
    password = input("Enter password: ")
print("Access granted!")

// Infinite loop with break
while True:
    command = input("Enter command: ")
    if command == "quit":
        break
    process(command)

Repeat Loops (Carrion Special)

Carrion's unique way to repeat code a specific number of times:

// Repeat 5 times
repeat 5:
    print("Hello!")

// Useful for simple repetition
repeat 3:
    print("Hip hip hooray!")

// Nested repeat
repeat 3:
    repeat 4:
        print("*", end="")
    print()
// Output:
// ****
// ****
// ****
Tip: Use repeat when you just need to execute something N times without needing the loop counter.

Loop Control

Break

Exit a loop immediately:

// Find first even number
numbers = [1, 3, 5, 4, 7, 9]
for num in numbers:
    if num % 2 == 0:
        print(f"Found even: {num}")
        break

// Exit nested loops (breaks innermost)
for i in range(10):
    for j in range(10):
        if i * j > 25:
            print(f"Breaking at {i}, {j}")
            break

Continue

Skip to the next iteration:

// Skip odd numbers
for i in range(10):
    if i % 2 != 0:
        continue
    print(i)  // 0, 2, 4, 6, 8

// Skip invalid entries
for item in items:
    if item == None:
        continue
    process(item)

Error Handling Flow

Attempt / Ensnare / Resolve

Handle errors gracefully:

attempt:
    // Code that might fail
    file = File.open("data.txt", "r")
    content = file.read()
    process(content)

ensnare FileNotFoundError:
    print("File not found!")

ensnare PermissionError:
    print("Permission denied!")

ensnare as e:
    // Catch any other error
    print(f"Error: {e}")

resolve:
    // Always runs, even if there was an error
    print("Cleanup complete")

Raising Errors

spell divide(a, b):
    if b == 0:
        raise ValueError("Cannot divide by zero!")
    return a / b

attempt:
    result = divide(10, 0)
ensnare ValueError as e:
    print(e)

Pattern Examples

Guard Clauses

Return early for cleaner code:

spell process_user(user):
    if user == None:
        return "No user provided"

    if not user.is_active:
        return "User is inactive"

    if not user.has_permission:
        return "Permission denied"

    // Main logic here
    return do_processing(user)

Accumulator Pattern

// Sum all numbers
numbers = [1, 2, 3, 4, 5]
total = 0
for num in numbers:
    total = total + num
print(total)  // 15

// Build a string
words = ["Hello", "World"]
sentence = ""
for word in words:
    sentence = sentence + word + " "
print(sentence.strip())  // "Hello World"

Search Pattern

spell find_item(items, target):
    for i in range(len(items)):
        if items[i] == target:
            return i
    return -1  // Not found

index = find_item(["a", "b", "c"], "b")
print(index)  // 1
